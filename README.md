# @awaken/taxsdk

Tax SDK for Awaken

## Installation

```bash
npm install @awaken/taxsdk
```

## Usage

```typescript
import { TaxCalculationOptions, TaxCalculationResult } from '@awaken/taxsdk';

// Your usage example here
```

## Development

```bash
# Install dependencies
npm install

# Build the project
npm run build

# Watch mode
npm run build:watch

# Type checking
npm run typecheck

# Clean build artifacts
npm run clean
```

## Publishing

```bash
# Build and publish to npm
npm publish
```

The `prepublishOnly` script will automatically build the project before publishing.
